โดย website นี้มีการใช้ภาษาทั้งหมด 3 ภาษา ได้แก่ HTML CSS JAVASCRIPT โดยมีหลักการทำงานก็คือ 

1. การทำโครงสร้างของ website ใช้ ภาษา HTML
2. ตกแต่งและทำ responsive โดยใช้ CSS
3. ทำ action กับผู้ใช้งานโดยใช้ JAVASCRIPT

โดยจะมีการเขียนโครงสร้างโดยใช้ HTML ขึ้นมาก่อนหลังจากนั้นใช้การสร้าง class ของ HTML เพื่อนำไปเชื่อมกับ CSS สำหรับการตกแต่ง และมีการใช้ id เพื่อนำไปทำ action โดยใช่ JAVASCRIPT
